# Appendix A — Glossary

- **Intent Probability Wave (IPW):** การกระจายความน่าจะเป็นเหนือชุดการกระทำที่เป็นไปได้
- **Ghost Action:** การแสดงผลล่วงหน้าเชิงคาดการณ์แบบถอนได้
- **Manifestation:** การปรากฏของ state ภายในเป็นภาษาแสง
- **Honest Incompetence:** เมื่อระบบไม่มั่นใจ ต้องแสดงความไม่แน่นอนอย่างซื่อสัตย์
- **Deterministic Synchronization:** โหนดหลายตัวให้ผลเหมือนกันจาก state/input เดียวกัน
- **Wave Collapse:** จุดที่ action จริงยืนยัน prediction เพื่อ commit หรือ rollback
